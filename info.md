## Icon Explorer Card

Search and browse **18,000+ icons** from all your installed icon packs with live search, pagination, and one-click copy.

### Supported Icon Packs
- Material Design Icons (`mdi:`)
- Hass Hue Icons (`hue:`)
- Custom Brand Icons (`phu:`)
- Simple Icons (`si:`)
- Fluent UI System Icons (`fluent:`)
- IconPark (`icon-park:`)
- SVG Logos (`logos:`)
- Weather Icons (`wi:`)

### Quick Start
```yaml
type: custom:icon-explorer-card
title: "Icon Explorer"
```

### Features
- Live search across all packs
- Paginated loading (no browser freeze)
- One-click copy to clipboard
- 24h localStorage caching
- Fully configurable
